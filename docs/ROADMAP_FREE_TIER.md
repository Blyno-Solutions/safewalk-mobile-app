# 🗺️ Free Tier Open-Source Roadmap (Batch 2026)

Welcome to the Blyno Solutions Open-Source Internship. To qualify for a certificate, you must successfully complete and merge 3 Pull Requests (PRs) following this timeline.

## 📜 The Golden Rules

1. **Self-Serve:** Mentors do not provide guidance for this tier.
2. **Quality First:** PRs must pass the **80% Test Coverage** CI check.
3. **No Duplicates:** Check existing PRs before starting. We only merge the _best_ implementation.
